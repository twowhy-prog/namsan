# Namsan 프로젝트

## 프로젝트 개요
남산 터널 혼잡도 실시간 안내 웹앱 (단일 HTML 파일, `index.html`)

## 구현 예정 기능

### 원베일리 → 고덕로210 경로 안내 (미구현)
**출발지:** 원베일리 (반포동)  
**목적지:** 고덕로210

두 가지 경로를 비교해서 빠른 길 안내:
1. **올림픽대로** 경유
2. **강변북로** 경유

현재 `TUNNELS` 배열에 해당 경로 데이터가 없음. 구현 시 Kakao Mobility API 및 서울 교통 API 연동 필요.

## 현재 구현된 경로
- 출발지: 삼일대로330 (을지로입구역)
- 목적지: 원베일리 (반포동)
- 경로: 남산 1호 터널, 2호 터널, 3호 터널 비교

## 기술 스택
- 단일 HTML 파일 (JavaScript + CSS 포함)
- Kakao Mobility Directions API
- 서울 교통정보 API
- 30초 자동 새로고침
